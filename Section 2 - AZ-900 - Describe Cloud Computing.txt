3 Main Types of cloud environment

Public Cloud

	Is defined as computing services offered by third-party providers over the public Internet, making them available to anyone who wants to use or purhcase them.
	Azure owns the hardware on their network and infrastructure.
	
Private Cloud

	Is defined as computing services offered either over the Internet or a private internal network and only to selectt users instead of the general public.
	Looks and acts like a cloud, except customer owns or leases or has exclusive access to the hardware.
	Example: Azure Government Cloud
	
	Azure Local (Formerly Azure Stack HCI) - Lets you run azure in your private hardware.
	
	Azure Arc - enables you to manage and govern resources across on-premises, multi-cloud, and edge environments as if they were running in Microsoft Azure.

Hybrid Cloud

	Is a computing environment that combines a private cloud with a public cloud.
	Combination of public and private clouds; scale private infrastructure to the cloud.
	
	Azure File Sync - a service that allows you to centralize your organization’s file shares in Azure while keeping the flexibility, performance, and compatibility of an on-premises file server.

Cloud Pricing

Factors affecting VM Pricing

	- Geographical Region
	- Instance Size
	- Bandwidth
	- Reservation/Saving Plan
	- Operating System SKU / License
	- Disk Type / Size
	- Backup Storage
	- Support Aggrement
	
Factors Affecting Cosmos DB Pricing

	- API Choice
	- Region/Regions
	- Standard Model/ Serverless
	- Number of Operations per second
	- Consumed Storage
	- Optional Dedicated Gateway
	- Backup Storage
	
Payment Categories

You can Pay for time
	- Some Serives charge by time used.
	- Stop using the service to stop the charges.
	- ie. Virtual Machine instances.
	
You can Pay for GB
	- Usually you pay for any data storage used.
	- Fairly cheap - 0.02 per GB per month
	- Also pay for network traffic to/from other regions, or to the internet.
	- Inbound Traffic is free/ Outbound is billed

Pay for Operations
	- In addition to paying for storage space, and bandwidth, you can also be charged per operation.
	- Operation are read, write, list, delete
	- Pay per message, pay per query to a database, etc.
	- Usually very cheap per operation.
